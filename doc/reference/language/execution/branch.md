# ngl:lang:execution:branch
>ngl:branch
{
    (_branch_expression_) _concrete_expression_ `list` 
}

Example:
```
ngl:function divide
{
    ngc::number <a>
    ngc::number <b>
    
    ngl:branch
    {
        (a != 0) .result = a / b
        () ngl:terminate 
    }
}
```
